# chatbot-firebase

A simple chatbot application built with **FastAPI**, **Firebase Authentication**, and **Firestore**.

## 🚀 Features

* Lightweight chat UI
* **User authentication before chatting**
* **Save and load chat messages**
* Easy to extend with any API or AI model

## 🧰 Requirements

* Python 3.x
* FastAPI
* Firebase Admin SDK
* Other dependencies in `requirements.txt`

## ⚙️ Installation

```bash
git clone https://github.com/donghailam/lab-2.git
cd lab-2
py - m pip install -r requirements.txt
```

## ▶️ Run

```bash
cd backend
uvicorn main:app --reload
```

```bash
cd frontend
python -m http.server 5500
```

## 🛠️ Customization

Edit files to:

* Change UI
* Modify API logic
* Connect to AI services
* Store data in database
